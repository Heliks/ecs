Simple data oriented [Entity Component System](https://en.wikipedia.org/wiki/Entity%E2%80%93component%E2%80%93system)

WIP

# Install

```shell
$ npm install --save @heliks/ecs
```

# Development

## Requirements

- Node 14

Install pnpm if it is not already installed on your system: 

```bash
$ npm install -g pnpm
```

## Building

```bash
$ pnpm build
$ pnpm build:watch
```
